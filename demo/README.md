# Demo

Walkthroughs of TeamDesk's THE 3 + AI skills, by role.

## TEAM member (`demo-team.gif`)
Sign in → `/desk` THE 3 (read-only banner + status pills, no admin controls) →
mark an item done-to-perfection → Focus Log tab → `/team` calendar.

## ADMIN (`demo-admin.gif`)
`/desk` with full controls → add a focus item → **Sharpen with AI** (DeepSeek
critique + rewrite) → Apply → **Review today's 3 (CEO)** → **Office Hours** advice
→ **Weekly review**.

The role difference is the point: TEAM gets the read/status surface; ADMIN gets
editing plus all four AI skills.
